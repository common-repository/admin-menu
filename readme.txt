== Installation ==
**Normal installation**

1. Download the admin-menu-editor.zip file to your computer.
1. Unzip the file.
1. Upload the `admin-menu-editor` directory to your `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
That's it. You can access the the menu editor by going to domain.com/admin.php?page=admin-menu-pro. The plugin will automatically load your current menu configuration the first time you run it.
